# AI Blockchain Project

## Project Title
AI Blockchain Project for Academic Certificate Verification

## Project Description
This project is developed as part of the U2U Internship Program.

The project uses Blockchain Technology and Smart Contracts to securely issue, store, and verify academic certificates. It helps prevent fake certificates and allows quick verification in a secure and transparent way.
